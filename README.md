# 陈其敏｜数据分析作品集

已通过 GitHub Pages 发布的数据分析与科学建模作品集。

## 本地查看

直接双击 `index.html`，或在浏览器中打开该文件。首页的三个项目封面均可进入独立详情页。

运行 Python 源码前可执行 `pip install -r code/requirements.txt`。A/B 脚本也内置 SciPy 回退，即使暂未安装 `statsmodels` 仍可复算 Z 检验与置信区间。

## 内容结构

- 广告投放 A/B 实验：数据质量、指标口径、分层诊断、比例 Z 检验、投放建议。
- 英国电商分析：订单清洗、经营指标、市场拆解、RFM 分层、运营建议。
- 手机电池建模：问题定义、特征体系、电热耦合模型、敏感性分析、情景模拟与优化。
- 每个项目均包含 7 步可点击流程、交互图表、Python 源码和 MySQL 源码。

## 数据与展示说明

- A/B 与电商项目的核心指标来自用户提供的原始 CSV，并经过独立复算。
- 主视觉采用 Tableau 风格的信息层级；页面中的数字、标注和可交互图表均由代码按照复算结果绘制。
- 当前未提供电池项目原始数据，因此该页清楚标注为机制演示，不把模拟结果包装成真实业务结论。
