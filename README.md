# 数据分析作品集网站

## 使用方式

直接双击 `index.html` 即可浏览。网站为纯静态实现，不需要安装依赖或启动服务。

## 页面结构

- 首页：个人定位、项目轮播与分析方法
- 项目 01：广告投放 A/B 实验与用户转化分析
- 项目 02：英国电商用户行为与 RFM 用户价值分析
- 项目 03：手机电池续航影响因素分析与优化

每个详情页包含真实项目规模指标、业务问题、六步分析链、可交互数据组件、核心代码、成果总结和分析边界。

## 本次原始数据复算

- 广告实验：`marketing_AB.csv`，588,101 名独立用户。实验组转化率 2.5547%，PSA 对照组 1.7854%，绝对提升 0.7692 个百分点，相对提升 43.09%。
- 英国电商：`data.csv`，541,909 条原始明细。清洗后保留 392,692 条有效交易、18,532 笔订单和 4,338 名客户，有效销售额 £8.887M。
- 手机电池：当前未提供对应原始数据，因此页面中的模拟结果只用于解释模型机制。

图表中同时展示必要的样本量、时间范围或清洗口径。2011 年 12 月仅包含 1–9 日；曝光频次和转化率之间只解释为相关关系。

## 修改入口

- 项目内容和交互逻辑：`app.js`
- 页面结构：`index.html`
- 视觉样式：`styles.css`、`overrides.css`
- 项目封面：`assets/`
- 完整分析代码：`source/`

## 完整源代码

- `source/01_ab_analysis.py`：质量检查、宽表、分箱、Z 检验、置信区间和分层指标
- `source/01_ab_mysql.sql`：MySQL 清洗宽表和实验指标体系
- `source/02_ecommerce_rfm.py`：订单清洗漏斗、订单级聚合、经营指标和 RFM 分群
- `source/03_battery_model.py`：功耗模块、SOC—温度耦合 ODE、场景仿真和约束优化

A/B 与电商页面已经使用本次提供的原始 CSV 复算。电池页面未提供原始文件，所有模型演示值均已明确标注。发布前请按需补充联系方式。
